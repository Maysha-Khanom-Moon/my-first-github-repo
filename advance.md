    #### git branch

        * (*main --> on main branch)

    - git checkout main
        * (main branch a switch korbe)

    - git checkout -b moon
        * (moon nam a ekta new branch open hobe)

    - new branch a push korte gele ektu genjam korbe. pashapashi appropiate command show korbe
